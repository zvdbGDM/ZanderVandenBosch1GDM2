Over Mij

Ik ben Zander Van den Bosch van 1GDM2. Deze website is mijn portfolio voor het vak Prototyping, waar ik mijn reis van 13 weken documenteer.

Tijdens dit vak heb ik leren werken met applicaties zoals Tailwind CSS, JavaScript, Studio Visual Code en andere verschillende tools. We kregen wekelijks nieuwe opdrachten die verder gingen op wat we in de les hadden gezien en moesten met de nieuwe dingen die we zagen in deze les anpassingen maken aan het vorige design.

In deze portfolio is mijn groei als developer en designer zichtbaar.
